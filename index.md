# 内容概述

Android Cars文档整理，更准确地说是Android Automotive OS

Android Auto不开源，不在范围，有兴趣可以参考 [官方文档](https://developer.android.com/training/cars#automotive-os).

2019年基于AOSP做了汽车HMI的开发，再次做一下梳理。

可能包含的主题（更新可能比较慢）：

- 架构
- 属性
- 显示/输入
- HMI
- 连接
- 音频
- 相机
- 媒体
- 多用户
- 安全
- 测试
- ...
